Get started

What you need to do

  1. Sign up to get your free credits.
  2. Copy-paste the code snippet to call our API.
  3. Get a subscription to switch to production.

How it works

  • The code snippet is a simple HTTPS request that sends the CV to our API.
  • The CV is parsed, and its extracted data is directly returned in the response of the HTTPS request.
  • Store the extracted data in your own database, because we instantly delete it from our servers once the parsing is done.

Details

  • Accepted CV content: in any human language.
  • Accepted CV formats: .pdf, .docx, .doc, .odt, .pptx, .ppt, .jpeg, .png.
  • You can make multiple API calls in parallel to parse up to 500 CVs/minute.
  • Contact us if you have this type of high volumes.
